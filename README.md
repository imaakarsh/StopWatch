# Stopwatch and Timer

A polished React + TypeScript stopwatch and countdown timer built with Vite.

## Features

- Stopwatch with start, pause, and reset controls
- Timer with custom hours, minutes, and seconds input
- Clear, responsive time display with accessible controls
- Mobile-friendly two-panel layout with polished styling

## Run Locally

```bash
npm install
npm run dev
```

## Available Scripts

- `npm run dev` starts the Vite development server
- `npm run build` creates a production build
- `npm run lint` runs ESLint across the workspace

## Notes

The timer keeps the configured duration in sync with the input fields when it is not running. Reset returns the display to the currently configured time.
